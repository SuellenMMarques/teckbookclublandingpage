# 📚 Tech Book Club Landing Page

Projeto desenvolvido a partir de um desafio do Frontend Mentor, com foco na construção de uma landing page moderna, responsiva e bem estruturada.

## 🚀 Sobre o projeto

Esta é uma landing page para um clube de leitura voltado para desenvolvedores. O objetivo é apresentar o serviço, destacar benefícios e incentivar a conversão através de planos de assinatura.

O layout foi fielmente implementado com atenção à responsividade, organização de código e boas práticas de desenvolvimento front-end.

## 🛠️ Tecnologias utilizadas

* HTML5 semântico
* CSS3 (com variáveis e organização modular)
* JavaScript (manipulação de DOM)
* Mobile First
* Flexbox e Grid

## 🎯 Funcionalidades

* Layout totalmente responsivo (mobile, tablet e desktop)
* Estrutura semântica e organizada
* Componentização visual com reutilização de classes
* Injeção dinâmica de elementos (estrelas de avaliação via JavaScript)
* Uso de `<picture>` para imagens responsivas

## 🧠 Aprendizados

* Estruturação de layout com abordagem mobile-first
* Uso avançado de variáveis CSS para padronização
* Combinação de Flexbox e Grid para layouts complexos
* Manipulação do DOM com JavaScript de forma simples e eficiente
* Organização de código pensando em escalabilidade

## 💡 Pontos fortes do projeto

* Código limpo e bem organizado
* Boa separação de responsabilidades (HTML, CSS e JS)
* Uso consistente de design tokens (cores, espaçamentos, tipografia)
* Responsividade bem aplicada
* Pequena interação com JavaScript sem exagero
